# Personal Portfolio Website

A responsive personal portfolio website developed to present my technical skills, projects, and professional background as an Electronics and Communication Engineering student with a focus on software development.

## 🔗 Live Demo

- https://chdeekshith14.github.io/My-Personal-Portfolio/

## Overview

This portfolio serves as a centralized platform to showcase my work, including academic projects, independent builds, and technical competencies. It is designed with a clean user interface, responsive layout, and smooth navigation to ensure accessibility across devices.

---

## Tech Stack

- HTML5 – Semantic structure and content organization  
- CSS3 – Styling, layout, and responsive design  
- JavaScript – Interactivity and dynamic behavior  

---

## Key Features

- Fully responsive design for mobile, tablet, and desktop  
- Dark-themed modern UI with clean visual hierarchy  
- Smooth scrolling navigation between sections  
- Interactive UI elements for improved user experience  
- Organized sections for About, Skills, Projects, and Contact  
- Lightweight and optimized for fast loading
